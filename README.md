Movie Rating Prediction & Recommendation System

Overview:-
This project implements a supervised machine learning model to predict
movie ratings based on genre and duration, and recommends top movies
using predicted ratings.

Concepts Used:-
- Pandas & NumPy for data handling
- One-Hot Encoding for categorical features
- Linear Regression (Supervised Learning)
- Model evaluation using Mean Squared Error
- Prediction-based recommendation system

Tools & Libraries:-
- Python
- Google Colab
- Scikit-learn
- Pandas, NumPy

Features:-
- Predict ratings for unseen movies
- Recommend top 5 movies based on predictions
